# Notas de Release

## Título sugerido

`FINAL FANTASY XII THE ZODIAC AGE - Tradução PT-BR`

## Descrição curta

Tradução em português brasileiro para FINAL FANTASY XII THE ZODIAC AGE, com uma versão automática recomendada e uma versão manual alternativa.

## Corpo sugerido

Esta release contém duas formas de instalação:

### 1. Versão automática `r1` (recomendada)

- instalador automático
- fluxo mais simples para o usuário final
- melhor opção para a maioria das pessoas
- vídeo de instalação: [YouTube](https://www.youtube.com/watch?v=QzFkaFCr_fI)

### 2. Versão manual `r3` (alternativa)

- usa `FF12 External File Loader`
- pacote sem executável
- indicada para quem prefere instalar manualmente

### Instalação da `r1`

1. Extraia `FF12_PTBR_Installer_v1.8.0-r1.zip`.
2. Execute `FF12_PTBR_Installer_v1.8.0-r1.exe`.

### Instalação da `r3`

1. Instale o `FF12 External File Loader`.
2. Confirme que existem `x64\dinput8.dll` e `x64\modules\ff12-file-loader.dll` na pasta do jogo.
3. Extraia `FF12_PTBR_v1.8.0-r3.zip`.
4. Copie `ff12data` para `mods\deploy`.
5. Confira se o caminho final ficou `mods\deploy\ff12data`.

### Integridade

Os hashes SHA-256 dos assets estão em `docs/ASSET_HASHES.md`.
